# Desafio 1
Esse programa tem como objetivo ajudar o usuário a decidir se uma viagem é viavel ou não de acordo com um orçamento previamente estabelecido.
Ao rodar o desafio1.py, o programa irá solicitar ao usuário os dados: 
1. Orçamento disponível (reais).
2. Destino da viagem.
3. Custo da passagem (reais).
4. Custo diário da hospedagem (Euros).
5. Quantidade de dias.
   
Depois disso, o programa irá mostrar ao usuário se a viagem é viavel e quanto do orçamento sobrará/faltará.

Perguntas teóricas:

■ Qual a diferença entre o comando git add . e git commit -m "mensagem"?

- O comando git add . prepara os arquivos modificados para serem incluidos no proximo commit. Já o git commit -m cria um registro permanente das alterações e salva como uma nova versão do projeto.

■ Por que é necessário realizar o casting (conversão de tipo) ao usar a função input() em Python para cálculos matemáticos?

- Porque sem essa conversão o Python trataria os dados de entrada como tipo texto (String) e não como valores numéricos (Int/Float)

■ O que acontece se tentarmos somar uma variável do tipo str com uma do tipo float

- O Python gera um erro do tipo (TypeError).
